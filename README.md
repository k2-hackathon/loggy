## Getting Started

### Prerequisites

- Node.js 16.13.0 https://github.com/nodenv/nodenv
- Yarn https://yarnpkg.com/
- Docker Desktop https://docs.docker.com/desktop/install/mac-install/

for Backend 

- poetry https://cocoatomo.github.io/poetry-ja/

### Installing

```shell
yarn
```

```shell
docker-compose up -d --build
```
